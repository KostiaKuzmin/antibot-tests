Вам предоставлен антибот-сервер, который анализирует HTTP-запросы и определяет результат,
является ли клиент ботом, антидетект-браузером или реальным пользователем.

Изучите код сервера и сформулируйте 5-7 тест кейсов.

Нужно что бы присутствовали следующие типы тестов
а) Позитивный тест (реальный браузер)
б) Негативный тест (бот)
в) Пограничный тест (подозрительное поведение)

НУЖНО написать автотесты с использованием python, pytest, (playwright или selenium)

По желанию можно упаковать в docker контейнер.
